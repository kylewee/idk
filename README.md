# IDK - Decision Making Helper

A simple command-line tool to help you make decisions when you just don't know!

## Features

- Choose randomly from a list of options
- Coin flip decisions
- Yes/No decisions  
- Magic 8-ball style responses

## Usage

```bash
# Choose from options
python main.py pizza burger tacos

# Flip a coin
python main.py --coin

# Get a yes/no answer
python main.py --yes-no

# Ask the magic 8-ball
python main.py --8ball
```

## Requirements

- Python 3.6 or higher
- No external dependencies required

## Examples

```bash
$ python main.py coffee tea
IDK, but I choose: coffee

$ python main.py --coin
Coin flip result: Heads

$ python main.py --8ball
Magic 8-ball says: Signs point to yes
```
