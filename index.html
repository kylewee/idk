About This Deployment

This site demonstrates the integration between GitHub Actions and Netlify using the following secrets:

  • NETLIFY_AUTH_TOKEN - Used to authenticate with Netlify API
  • NETLIFY_SITE_ID - Identifies the specific Netlify site to deploy to

The deployment is automatically triggered when code is pushed to the main branch.
